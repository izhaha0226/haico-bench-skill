# HAICo Bench Prompt Sequence

## 1. Divergence
Give me 7 clearly different directions for this creative task.
Do not create the final deliverable yet.
Use remote analogies such as myth, history, memes, pop culture, inverse framing, and category-breaking angles.
For each direction provide:
- name
- core metaphor/worldview
- one-line hook
- emotional tone
- best-fit channel
- risk

## 2. Benchmark mapping
For each direction, map which benchmark patterns it aligns with:
- hook pattern
- CTA pattern
- tone pattern
- format pattern
Do not copy benchmark wording. Only reference structure.

## 3. Selection
Evaluate all directions on:
- distinctiveness
- brand fit
- execution ease
- scalability
Return top 2 only.

## 4. Convergence
Refine only the #1 direction into a final deliverable.
Do not blend unrelated directions unless explicitly requested.

## 5. Anti-fixation check
Before finalizing, answer:
- Did we improve the first plausible idea, or did we truly explore?
- Are the chosen ideas meaningfully different from each other?
- Did benchmark usage stay structural rather than imitative?
