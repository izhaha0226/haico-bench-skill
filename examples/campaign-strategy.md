# Example: Campaign Strategy Workflow

## Task
Create campaign concept directions that can scale across multiple assets.

## Divergence prompt
Generate 7 campaign directions.
Do not write the final strategy deck yet.
For each direction provide:
- direction name
- campaign worldview
- core hook system
- proof logic
- CTA philosophy
- benchmark-fit note
- risk

## Selection prompt
Evaluate each direction on:
- distinctiveness
- scalability
- strategic coherence
- benchmark-fit quality
- derivative risk
Return top 2 only.

## Convergence prompt
Refine only #1 into:
- campaign thesis
- 3-message architecture
- asset system
- hero hook family
- CTA rhythm
- benchmark safety note

## Anti-fixation check
Before finalizing, answer:
- Is this truly campaign-worthy or just a strong single asset idea?
- Did we choose the most expandable system, not just the prettiest hook?
