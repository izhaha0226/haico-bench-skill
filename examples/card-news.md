# Example: Card News Workflow

## Task
Create a benchmark-aware card-news concept without locking into the first layout idea.

## Divergence prompt
Generate 5 different card-news concepts for this topic.
Do not generate final slide copy yet.
For each concept provide:
- concept name
- narrative arc
- slide rhythm
- hook strategy
- CTA strategy
- benchmark-fit note
- risk

## Selection prompt
Rank the concepts by:
- distinctiveness
- benchmark-fit
- narrative clarity
- conversion potential
- production ease
Return top 2 only.

## Convergence prompt
Develop only the #1 concept into a 6-slide structure:
- slide goal
- slide headline
- slide body idea
- visual direction
- CTA placement

## Anti-fixation check
Before finalizing, answer:
- Did we choose the strongest narrative or simply the first viable one?
- Is benchmark influence structural, not imitative?
- Would a human reviewer perceive clear conceptual originality?
