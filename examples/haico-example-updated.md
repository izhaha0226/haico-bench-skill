# Example: HAICo Worked Example

This example comes from an OpenClaw/Chief-style practical HAICo exercise and is kept as a compact worked example.

## Input
- Topic: 스마트폰 사용 줄이기 포스터
- Goal: attention-grabbing, memorable, action-oriented

## Divergent mode
Generate 5 distinct directions:
1. 방해하는 소(Interrupting Cow) 밈
2. 붉은 실로 손을 묶는 경고형 콘셉트
3. 스마트폰이 자석처럼 사람을 끌어당기는 콘셉트
4. 눈부신 네온과 알림 폭주 콘셉트
5. 손에서 빠져나가는 모래시계 콘셉트

## Why this divergence works
- each direction changes the metaphor, not just the wording
- the emotional engine changes: humor, warning, attraction, overload, loss
- the visual system changes with the concept

## Selection
- Pick: 붉은 실로 손을 묶는 경고형 콘셉트
- Why: 직관적이고 메시지가 단순하며 강한 경고 톤을 만든다.

## Convergent mode
### Title
- 폰이 아니라 당신의 시간이 필요합니다

### Subtitle
- 알림은 잠깐, 집중은 오래

### CTA
- 오늘 1시간만 꺼두세요

## Visual direction
- Black + red + white
- Smartphone wrapped with a red thread
- Strong, warning-style composition

## Delivery rule
- Never stop at the first result.
- Always generate multiple directions before choosing.
- Use remote analogies to break fixation.

## Practical takeaway
This example is intentionally simple.
Its job is to show the minimum viable HAICo loop:
- clear input
- multiple directions
- explicit selection
- one chosen result
- visual direction attached to concept
- delivery rules preserved
